# My Alarm Clock

This is my **first** project, in which I plan to create a digital alarm clock app, before attempting to make a physical device!

Since this is my first time doing something like this, I plan to start with a simple command-line app, then make a Windows GUI app,
then eventually make something physical out of it.

I'm currently working on the command-line version.

<img width="1217" height="677" alt="image" src="https://github.com/user-attachments/assets/5de817f6-cd8a-40db-9b89-5835cc794a1c" />

## Current Features:

* Automatically refreshing the time (based on system time).
* Creating/Deleting alarms.
* Creating/Managing multiple alarms at once.

## Features I Plan to Add

* Actual alarm noises.
* Having the digital versions run in the background.
* Adding messages to alarms.
* Saving current alarms to a file for persistence on app close/open.
* Alarm snoozing.
